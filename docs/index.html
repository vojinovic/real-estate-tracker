Pretrage koje pratim

Dodaj novi oglas

Zalepi URL oglasa sa halooglasi.com. Tracker ce ga proveravati svaki put kad pokrenes ./run.sh.

Dodaj novu pretragu

Otvori halooglasi, podesi filtere (grad, cena, kvadratura), pa zalepi URL iz adresne linije. Tracker ce te obavestiti email-om kad se pojave novi oglasi koji odgovaraju toj pretrazi.

GitHub konfiguracija

Za dodavanje oglasa kroz dashboard treba ti GitHub Personal Access Token (PAT). Token se cuva samo u tvom browseru (localStorage), nije nigde poslat.

Token napravi na: github.com/settings/tokens/new. Izaberi scope repo, klikni Generate, kopiraj token.
